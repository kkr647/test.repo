Ok… now back to buisiness: PYTHON AND R.

It’s no secret that both of these two are incredibly important for the data science world! Although many people have opinions on which is better, you can actually get these two to effectively speak to each other. Cool, huh?

If you’re working from R, all you need to do is run the “reticulate” library package in Rstudio. Do not forget to specify whether or not you’re using R or python using curly brackets. What is so amazing is that both of these codes from different sources like R and python can be used!

library(reticulate) 
# R code chunk
hi<-"HAPPY" 
hi
## [1] "HAPPY"
# Python Chunk 
hi="MOTHERS-DAY"
print(r.hi, hi)
## HAPPY MOTHERS-DAY
